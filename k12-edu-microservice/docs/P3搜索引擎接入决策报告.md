## 是否进入 P3

**是 — 执行 P3 影子接入**

当前数据量（480 条）未强制要求 P3，MySQL 主搜 P95/P99 仍在可接受范围。但为了后续资源规模增长、正文检索、拼音检索、中文分词、高亮和聚合分面，**执行 P3 影子接入**（OpenSearch 索引 `k12_search_document_v1`，双路对比，不立即替换 `/api/search/all`）。

| 配置项 | 初始值 |
|--------|--------|
| `k12.search.engine.enabled` | `false` |
| `k12.search.engine.shadow` | `true` |
| `k12.search.engine.fallback-to-mysql` | `true` |
| `k12.search.engine.provider` | `opensearch` |
| `k12.search.engine.index-name` | `k12_search_document_v1` |
