## 三、技术架构

### 前端技术栈
- Vue 3 + Composition API
- Element Plus UI
- Vue Router 4
- Axios

### 后端技术栈
- Spring Boot 3
- MyBatis-Plus
- Spring Security + JWT
- MinIO 对象存储

### 数据库
- MySQL 8.0
- MinIO（文件存储）

## 四、API 接口

### 用户端
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/resource/list | 资源列表 |
| GET | /api/resource/{id} | 资源详情 |
| GET | /api/resource/hot | 热门资源 |
| POST | /api/resource/search | 搜索资源 |
| POST | /api/resource/download | 下载资源 |
| POST | /api/resource/collect | 收藏资源 |
| DELETE | /api/resource/collect/{id} | 取消收藏 |
| GET | /api/resource/collections | 我的收藏 |

### 管理端
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/admin/resource | 创建资源 |
| PUT | /api/admin/resource/{id} | 更新资源 |
| DELETE | /api/admin/resource/{id} | 删除资源 |
| POST | /api/admin/resource/{id}/audit | 审核资源 |
| GET | /api/admin/resource/pending | 待审核列表 |
| POST | /api/admin/category | 创建分类 |
| GET | /api/admin/users | 用户列表 |
| PUT | /api/admin/users/{id}/status | 修改用户状态 |
| GET | /api/admin/stats/resources | 资源统计 |
| GET | /api/admin/stats/users | 用户统计 |

## 五、文件存储

### MinIO 配置
- Endpoint: http://localhost:9000
- Bucket: k12-resources
- Access Key: admin
- Secret Key: admin123

### 支持的文件格式
- 文档：PDF, DOC, DOCX, XLS, XLSX, PPT, PPTX
- 图片：JPG, PNG, GIF, WEBP
- 视频：MP4, AVI, MOV
- 音频：MP3, WAV, OGG

### 文件大小限制
- 单文件最大：100MB
