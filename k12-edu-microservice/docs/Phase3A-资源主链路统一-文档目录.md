# Phase 3A：资源主链路统一 - 文档目录

## 文档清单

1. `Phase3A-资源主链路统一-接口契约.md`
   - 统一 API 路径、请求参数、响应对象、兼容期策略

2. `Phase3A-资源主链路统一-字段字典.md`
   - 核心字段、文件域、维度域、挂载域、来源追踪定义

3. `Phase3A-资源主链路统一-枚举与状态机.md`
   - 双状态模型、枚举定义、状态流转、动作矩阵

4. `Phase3A-资源主链路统一-兼容映射与执行步骤.md`
   - 旧新接口映射、字段映射、执行顺序、门禁与回滚

## 推荐评审顺序

1. 先评审接口契约（确保路径和语义统一）
2. 再评审字段字典（确保数据结构统一）
3. 再评审枚举状态机（确保审核/发布规则统一）
4. 最后评审兼容映射与执行步骤（确保可落地可回滚）
