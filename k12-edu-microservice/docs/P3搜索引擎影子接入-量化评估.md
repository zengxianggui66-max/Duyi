## P3 影子接入路线图（建议）

```
Phase 1（现在）  MySQL 主搜 + 5 类报表 API
Phase 2          ES/OpenSearch 索引同步（与 sys_search_document 同源）
Phase 3          影子查询：/admin/shadow-compare?q=  返回 mysql vs es diff
Phase 4          灰度：10% 流量读 ES，MySQL 兜底
Phase 5          ES 主读，MySQL 仅降级
```

**暂不替换 MySQL**，直到：

1. 索引 > 1 万 **或** P95 > 500 ms 持续 7 天  
2. 零结果率（排除测试）< 8%  
3. CTR 样本 > 1000 次且 > 15%
