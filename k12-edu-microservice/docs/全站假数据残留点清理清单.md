## 2. 已关闭（原 P0 待清理项）

## T01 - 爱国教育页 mock 课程数据（已关闭）

- **文件**：`k12-edu-platform/src/views/feature/PatrioticEducation.vue`
- **原现象**：
  - 存在 `generateMockData()`
  - `allCourses = generateMockData()`
- **风险**：
  - 用户在真实环境仍可能看到假课程
- **处理结果**：
  - 已移除 mock 初始化入口
  - 当前为 `allCourses = ref([])` 空态策略，不再默认展示本地假数据
  - 后续待真实 API 接入后替换为线上数据源

## T02 - TopicResourceContent demoList fallback（已关闭）

- **文件**：`k12-edu-platform/src/components/subject/TopicResourceContent.vue`
- **原现象**：
  - `v-else-if="demoList.length"` 分支直接渲染演示列表
- **风险**：
  - 一旦上层误传 demoList，页面再次出现假数据
- **处理结果**：
  - 已移除 demoList 渲染分支
  - 组件仅保留真实 `apiResources` 与空态分支
