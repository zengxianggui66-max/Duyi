## 十一、数据优化规范

### 11.1 标题规范

```
小学语文一年级上册同步备课识字1春夏秋冬课件
初中数学七年级下册期中测试卷
高中语文必修上册赤壁赋教学设计
```

### 11.2 资源标签必填

学段、学科、年级、册别、版本、栏目、资源类型、关键词；特色频道资源必须打 `channel_key`。

### 11.3 索引规则

无 `route_path` 不应进入 `sys_search_document`。
