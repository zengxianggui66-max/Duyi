# 资源浏览页已完成改造清单

> 更新时间：2026-06-12  
> 目的：用于快速确认「已经完成了什么」，便于后续验收与继续迭代。

---

## 一、页面筛选与布局改造（已完成）

### 1. 国学阅读 / 作文筛选条重构
- 已将国学阅读筛选维度统一为：`年级 / 版本 / 单元 / 学期`。
- 已将筛选项改为单行布局，避免分散展示。
- 新增并接入 `ReadingWritingFilterBar.vue`，作为可复用筛选组件。

### 2. 主筛选统一为下拉形态
- 扩展筛选中的 `年级 / 学期 / 年份 / 版本 / 地区 / 类型` 已改为下拉菜单。
- 筛选区保持单行可读，不再多行堆叠。

### 3. “更多栏目”能力升级
- 已从简单折叠区升级为抽屉面板。
- 已支持关键词搜索栏目。
- 已支持最近使用栏目记忆（本地持久化）。
- 已支持热门栏目展示（基于真实数量）。
- 已支持分组计数与栏目计数展示（来自真实后端统计）。
- 已支持栏目排序优化（同组内按数量降序）。
- 已支持搜索命中高亮。
- 已支持「我的置顶」：
  - 置顶/取消置顶
  - 拖拽排序
  - 清空置顶
  - 恢复默认推荐

### 4. 栏目与类型联动纠正
- 切换栏目时，若当前类型不在目标栏目允许范围内，会自动重置为 `全部`。
- 同步备课类型已做收敛调整，去除不应混入的试卷类型。

## 二、资源列表与可见性体验（已完成）

- 删除了 `纯素材` 按钮。
- 资源卡操作区（如 `立即下载`、`+资料篮`）改为默认可见，不再仅 hover 可见。
- 术语统一：`资料蓝` 已统一修正为 `资料篮`。
- 多处页面乱码文本已清理为正常中文展示。

- 顶部功能顺序已调整为：`幼小衔接 / 小升初 / 中考 / 高考 / 对口升学`。
- 首页快捷功能中「课件资源」跳转逻辑已改为学段资源入口默认路径（与教育资源路径策略一致）。
