# 首页 → 学段资源页 功能测试用例

> 适用页面：`HomePage.vue`（首页）、`SubjectDetailPage.vue`（学段资源浏览页）  
> 路由：`/subject/:stage/:subject/:version`  
> 文档日期：2026-06-17  
> 默认环境：顶栏学段 **小学**；同步备课 **小学+语文**；试卷 **小学+六年级下册+期中**；升学 **幼小衔接+拼音识字**

---

## 1. 页面流转总览

```
首页 HomePage
  ├─ 顶栏 TopNavBar：学段切换 / 升学入口（直跳学科页）
  ├─ 左侧 LeftSidebar：选学科 → 打开 SubjectDetail 浮层
  │     ├─ 点「教材版本」→ 直跳学科页
  │     ├─ 点「资料类型」→ VersionVolumeSelector 弹窗 → 确认后跳转
  │     └─ 点「复习/升学栏目」→ 直跳学科页（带 module）
  └─ 专区模块：同步备课 / 试卷 / 升学 → 点资源进详情

学段资源页 SubjectDetailPage
  ├─ FilterBar：学科 / 栏目 / 更多栏目
  ├─ CourseCatalog：版本册别 / 目录树
  ├─ ResourceTypeBar：资源类型 + 角标
  ├─ AdvancedFilterBar：单份/成套 / 排序 / 批量下载
  └─ 资源列表 → /resource/:id
```

**标准落地 URL 示例：**

```text
/subject/primary/chinese/tongbian2024?module=同步备课&brand=qicai&volume={册别id}
```

## 4. 学段资源页部件功能点

### 4.1 顶部 `FilterBar`（`.filter-bar-container`）

| 行 | 功能 | 操作 | 预期 |

| 更多栏目 | 抽屉 | 打开选隐藏栏目 | 小升初/暑假等可选 |

**小学顶栏直出栏目：** 同步备课、国学阅读、期中复习、期末复习、开学专区、月考 + 更多(12)

### 4.2 左侧 `CourseCatalog`

| 功能 | 操作 | 预期 |
|------|------|------|
| 教材信息 | — | 显示学科·册别·版本 |
| 切换版本/册别 | 按钮 | 弹窗确认后刷新目录与列表 |
| 目录树 | 展开/折叠 | 仅 UI，不刷列表 |
| 选中节点 | 点击课文/单元 | 列表按节点过滤；URL `node` 同步 |
| 展开全部 | 按钮 | 树全部展开/收起 |

### 4.3 `ResourceTypeBar`（类型行）

| 功能 | 操作 | 预期 |
|------|------|------|
| 基础类型 Tab | 全部/课件/教案/练习… | 列表按类型过滤；角标为当前目录下该类型**总存量** |
| 「全部」角标 | — | = `/resources/browse/stats` 的 `total`，与列表 `total` 一致 |
| 0 条隐藏 | stats 就绪后 | 计数为 0 的类型 Tab **不展示**（「全部」始终展示，可为 0） |
| 「更多」 | 点击展开 | 展示小众类型（课堂实录、说课稿、逐字稿等）及角标 |
| 更多区 0 隐藏 | 展开后 | 0 条小众类型不展示；全部 0 时「更多」按钮不出现 |
| 点击类型 | 选课件/教案 | **仅过滤列表**，角标数字不变 |
| 目录切换 | 点左侧节点 | 重置为「全部」；角标与列表同步重算 |
| URL | — | 非「全部」时同步 `type` |

**统计口径：** 系列 + 学段 + 学科 + 册次 + 版本 + 栏目 + 左侧目录节点（`catalogNodeId` + `includeSubtree`）。

### 4.4 目录联动类型统计（CAT）

| 目录层级 | includeSubtree | 统计范围 |
|----------|----------------|----------|
| 课本根「语文（统编版）」 | true | 整册全部单元/课文 |
| 单元父节点「我上学了」 | true | 该单元下所有课文资源之和 |
| 课文叶子「我是中国人」 | false | 仅该课文匹配资源 |
| 特殊节点（附录/期中复习） | 按节点 meta | 该分类专属资源 |

**必测一致性：**

- `browse.total` === `stats.total` === 「全部」角标
- `stats.typeSum` === `stats.total`（P2 后端对账字段）
- 各类型 `displayType` 角标 === 列表该类型全量条数
- 分页切换时角标不变

### 4.6 P2：displayType 与目录 scope 表

| 层级 / nodeType | includeSubtree | 说明 |
|-----------------|----------------|------|
| root / textbook / depth=0 | true | 整册统计 |
| unit / folder / section | true | 单元含子课文 |
| lesson / leaf | false | 仅当前课文 |
| 未知 nodeType | 按 depth、`*_root`、clientHint 兜底 | 前后端 `CatalogBrowseScope` 一致 |

**displayType 映射（stats 分组 = Tab 名）：**

| Tab | API type | API subType |
|-----|----------|-------------|
| 精彩片段 | 视频 | 精彩片段 |
| 课文相关图片 | 图片素材 | 课文图片 |
| 其他 Tab | 同名 type | — |

### 4.7 `AdvancedFilterBar`

| 功能 | 操作 | 预期 |
|------|------|------|
| 找单份/找成套 | 切换 | 列表区切换单份/成套 |
| 综合/最新/下载量 | 排序 | 列表顺序变化 |
| 批量下载 | 开关 | 进入/退出多选模式 |
| 结果数 | — | 与 API total 一致 |
