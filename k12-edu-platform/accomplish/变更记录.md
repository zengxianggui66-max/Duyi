```java
/**
 * 撤回待审核资源（status: 0 → -1 草稿）
 */
public PrimaryChineseResource withdrawPending(Long id, Long userId) {
    // 1. 401 - 未登录检查
    // 2. 404 - 资源存在 + 未软删检查
    // 3. 403 - 上传者本人检查
    // 4. 400 - 仅 status=0(PENDING) 可撤回
    // 5. 设置 status=-1(DRAFT) + auditStatus=草稿 + publishStatus=未上架
    // 6. 同步 wide table / placement / search index
    return getById(id);
}
```

**防护规则**:
| 条件 | 返回 |
|---|---|
| 未登录 | `BusinessException(401, "请先登录")` |
| 资源不存在/已删除 | `BusinessException(404, "资源不存在")` |
| 非本人上传 | `BusinessException(403, "无权操作该资源")` |
| 状态非待审核 | `BusinessException(400, "仅待审核资源可撤回")` |

### 文件 4: PrimaryChineseResourceController.java

**改动**: 新增 1 个 API 端点 (L330-348)

```java
/**
 * 撤回待审核资源（status: 0 → -1 草稿）
 */
@PostMapping("/pending/{id}/withdraw")
public Result<PrimaryChineseResourceVO> withdrawPending(
        @PathVariable Long id,
        HttpServletRequest request) {
    Long userId = userIdResolver.resolve(request);
    if (userId == null) return Result.fail("请先登录");
    try {
        PrimaryChineseResource resource = primaryChineseResourceService.withdrawPending(id, userId);
        return Result.success(PrimaryChineseResourceVO.fromEntity(resource));
    } catch (BusinessException e) {
        return Result.fail(e.getCode(), e.getMessage());
    } catch (Exception e) {
        return Result.error("操作失败：" + e.getMessage());
    }
}
```

      <template #reference>

  <template v-else-if="item.status === 1">

## 数据流验证

```
POST /pending/{id}/withdraw
  │
  ├─ Controller.withdrawPending()
  │   ├─ userIdResolver.resolve(request)    → 获取用户ID
  │   ├─ 未登录 → 401
  │   └─ Service.withdrawPending(id, userId)
  │
  ├─ Service.withdrawPending()
  │   ├─ selectById(id)                     → 查资源
  │   ├─ 不存在/已删除 → 404
  │   ├─ 非本人 → 403
  │   ├─ 非待审核 → 400
  │   ├─ setStatus(DRAFT=-1)
  │   ├─ setAuditStatus(AUDIT_DRAFT=-1)
  │   ├─ setPublishStatus(PUBLISH_UNPUBLISHED=0)
  │   ├─ updateById()                       → 更新主表
  │   ├─ syncLegacyToWide()                 → 同步宽表
  │   ├─ syncPrimaryPlacement()            → 同步收录
  │   └─ afterPrimaryChanged()             → 更新搜索索引
  │
  └─ 返回 { id, status: -1, ... }
```
